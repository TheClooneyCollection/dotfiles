# Ghostty Config

## Files

- **config** -- main Ghostty config (hot-reloads on save)
- **themes.yaml** -- picked themes tagged light/dark with review progress (managed by [ghostty-theme-picker](../../Source/Projects/ghostty-theme-picker))
- **all-themes.txt** -- snapshot of all available theme names for diffing future Ghostty updates
- **classified-themes.yaml** -- auto-generated dark/light cache (regenerated by theme picker)

## Theme Picker

The theme picker is a separate project at `~/Source/Projects/ghostty-theme-picker/`.

```bash
ghostty-theme-picker    # browse and pick themes
ghostty-theme-generate  # bootstrap themes.yaml from curated list
```

## Checking for New Themes

After a Ghostty update, diff the bundled themes against the snapshot:

```
diff <(ls /Applications/Ghostty.app/Contents/Resources/ghostty/themes/) all-themes.txt
```

New entries are themes added since your last review. Update `all-themes.txt` after reviewing.
